# Mini Dashboard

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/SyedAhmad23/mini-dashboard.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd mini-dashboard
   ```

3. **Install the dependencies:**

   ```bash
   npm install
   ```

### Running the Application

To start up your high-speed Vite development server:

```bash
npm run dev
```
